# Дизайн док

### Ідея:
Планер задач з системою Pomodoro та елементами гейміфікації.
#### Фічі:
- Планування задачі з визначеними часовими рамками
	- створення задачі 
		- назва
		- опис
		- загальний  запланований час,
		- часовий помодоро інтервал
		- загальна кількість інтервалів
		- час на короткий відпочинок
		- час на довгий відпочинок
	- редагування задачі:
		- назва
		- опис
		- час інтервалів
		- кількість інревалів
		- час на відпочинок
	- редагування
	- зміна статусу
- Статистика виконаних задач протягом дня/тижня/місяця
	- інфографіка виконаних та невиконаних задач на день
	- стріки за щоденну активність
- Елементи гейміфікації
	- досвід за виконані задачі
	- додатковий досвід за стрік виконаних задач
	- механіка вихідних
	- різна складність задач - різна кількість досвіду

##### Основна функція:
Планування задач з визначеними часовими рамками.
*Наприклад задача - навчання 2 години.*
Кожна задача розбивається, за системою помадоро, на рівні проміжки часу.
*Наприклад, 2 г. / 20 хв. = 6*
Кожен часовий проміжок являє собою 1 кімнату данжу, або хп боса, наприклад
При завершенні всіх 6 "данжів" - перемога (виконана задача)
